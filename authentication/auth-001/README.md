# AUTH-001 — Modern SaaS Login Form

🔐 A production-ready, glassmorphism-styled login form built with vanilla HTML, CSS, and JavaScript.

## ✨ Features

- 🪟 **Glassmorphism Design** — Modern, frosted-glass aesthetic
- 🌙 **Dark Mode** — Midnight blue color palette
- 📱 **Fully Responsive** — Works on all screen sizes
- ♿ **Accessible** — Semantic HTML + ARIA labels
- ⚡ **Zero Dependencies** — Pure HTML, CSS, JS
- 🎯 **Focus States** — Clear visual feedback
- 🔘 **Social Login** — Google, GitHub, X buttons
- ✅ **Remember Me** — Custom checkbox
- 🔗 **Forgot Password** — Direct link
- 🎨 **Smooth Animations** — Subtle transitions & hover effects
