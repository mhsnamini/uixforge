# AUTH-001 — Ethereal Login Experience

A premium glassmorphism login interface built with pure HTML, CSS, and JavaScript.

Designed for modern authentication experiences with immersive visuals, smooth micro-interactions, real-time validation, and zero external dependencies.

![AUTH-001 Preview](preview.png)

---

## ✨ Highlights

* Glassmorphism UI with Midnight theme
* Real-time email validation
* Password strength indicator
* Floating label inputs
* Show / Hide password toggle
* Interactive particle background
* Custom animated cursor
* 3D card tilt effect
* Dynamic lighting and glare
* Social authentication buttons
* Toast notification system
* Success state animations
* Fully responsive layout
* Accessibility-friendly markup
* Zero dependencies

## ⚙️ Customization

### Theme Colors

```css
:root {
  --primary: #6c5ce7;
  --secondary: #00cec9;
  --accent: #fdcb6e;
  --bg-deep: #020617;
}
```

### Glass Effect

```css
.login-card {
  background: rgba(15, 21, 53, 0.55);
  backdrop-filter: blur(40px);
}
```

### Particle Density

```javascript
for (let i = 0; i < 65; i++) {
  particles.push(new Particle());
}
```
